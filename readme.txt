== Description ==

This plugin provides the view bulk action in the post, product, page or any custom post type in the admin section.

Plugin will add new bulk action name as "View" in you admin post view section.

Using this plugin you can view or open multiple posts in a one click with the new tabs.

== Key Features ==

1. Add bulk option in post, page. product & custom post type.
2. Open multiple post with bulk view option.

== NOTE ==

Disable the popup blocking in your browser to open multiple posts in new tab.

Share you thoughts/feedback on ajayghaghretiya007@gmail.com

== Installation ==

Add plugin "Bulk view post" in your site and just activate it. You will get the view option in the bulk option dropdown in any post type.
